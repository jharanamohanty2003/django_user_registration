# django_user_registration
A Django-based user registration and authentication system with login, signup, and secure password handling.
**Django User Registration System**

A secure and responsive user registration system built using the Django framework. This project includes user signup, login, logout, password validation, and authentication features with clean UI and proper form handling.

**Features**
User Registration
User Login & Logout
Password Validation
Authentication System
Django Forms Integration
Responsive Frontend
Error Handling & Messages
SQLite Database Support
